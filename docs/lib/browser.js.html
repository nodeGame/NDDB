NDDB browser routines to save and load items

Cyclic objects are decycled, and do not cause errors. Upon loading, the cycles are restored.

Loads a JSON object into the database from a persistent medium

Looks for a global store method to load from the browser database. The store method is supplied by shelf.js. If no store object is found, an error is issued and the database is not loaded.
